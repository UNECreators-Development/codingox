Database Quick Start¶

The following page contains example code showing how the database class is used. For complete details please read the individual pages describing each function.

Initializing the Database Class¶

The following code loads and initializes the database class based on your configuration settings:

$this->database();

Once loaded the class is ready to be used as described below.

Standard Query With Multiple Results¶ 

$query = $this->db->query('SELECT name, title, email FROM my_table');

foreach ($query->findAll() as $row)
{
        echo $row['title'];
        echo $row['name'];
        echo $row['email'];
}

The above findAll() function returns an array of standard array indexes. Example: $row[‘title’]

Standard Query With Single Result¶ 

$query = $this->db->query('SELECT name FROM my_table LIMIT 1');
$row = $query->findOne();
echo $row['name'];

The above findOne() function returns an array. Example: $row[‘name’]

Standard Insert¶

$sql = "INSERT INTO mytable (title, name) VALUES ("$title", "$name")";
$this->db->query($sql)->execute();
echo $this->db->affectedRow();

Query Builder Query¶

The Query Builder Pattern gives you a simplified means of retrieving data:

$query = $this->db->get('table_name');

foreach ($query->findAll() as $row)
{
    echo $row['name'];
}

The above get() function retrieves all the results from the supplied table. The Query Builder class contains a full compliment of functions for working with data.

Query Builder Insert¶

$data = array(
        'title' => $title,
        'name' => $name,
        'date' => $date
);

$this->db->insert('mytable', $data)->execute();
